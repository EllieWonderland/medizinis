# Datenarchitektur (Offline-First Relational)

## 1. UserSettings (Tabelle)
* `id` (PK)
* `global_herb_balance` (Integer)
* `day_cutoff_hour` (Integer, default: 4) // Einstellung für Nachteulen
* `created_at` (Timestamp)

## 2. Medications (Tabelle)
* `id` (PK)
* `name` (String)
* `reminder_times` (Array of Times)
* `package_size` (Integer)
* `current_stock` (Integer)
* `image_uri` (String, Pfad zum Beipackzettel-Foto)

## 3. Medizinis (Tabelle)
* `id` (PK)
* `species` (String / Enum)
* `gender` (Enum)
* `current_stage` (Enum: Egg, Baby, Child, Teen, Adult)
* `current_doses_progress` (Integer)
* `target_doses_for_next_stage` (Integer)
* `is_active` (Boolean)
* `is_retired` (Boolean, default: false)
* `is_unconscious` (Boolean)
* `last_medication_date` (Timestamp)

## 4. RoomState (Tabelle)
* `id` (PK)
* `medizini_id` (FK)
* `slot_floor_item_id` (String / Null)
* `slot_bed_item_id` (String / Null)
* `slot_wall_item_id` (String / Null)
* `slot_deco_item_id` (String / Null)
