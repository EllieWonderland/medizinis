# Projektübersicht: Medizinis

## Vision
Eine Gamification-App für neurodivergente Menschen (Fokus auf ADHS/exekutive Dysfunktion), die beim Aufbau einer verlässlichen Medikamenten-Routine hilft. Die App nutzt positive Verstärkung, verzichtet auf Bestrafungsmechaniken und agiert zu 100 % lokal (Offline-First, Datenschutz).

## Kernmechanik
Nutzer erhalten ein Ei. Durch die erfolgreiche Einnahme von Medikamenten ernten sie "Heilkräuter" und das Ei schlüpft nach 14 Einnahmen zu einem "Medizini" (einem magischen Tierwesen). Das Wesen durchläuft in einem Rhythmus von ca. 3 Monaten vier Entwicklungsstufen (Baby, Kind, Teenie, Erwachsener). 

## Tone of Voice & UI
* Kein klinisches Wording (statt "Medikament eingenommen" -> "Heilenergie gesammelt / Kräuter geerntet").
* Die Begleiterin und System-Stimme ist eine flauschige, freundliche Eule (Questgeber-Prinzip).
* Benachrichtigungen wecken den Fürsorge-Instinkt ("Dein Medizini hat Hunger"), ohne Schuldgefühle auszulösen.
